RABBIT

This is Rabbit, a free
EE library to
unify authentication.

  • Tell Me More

Rabbit is able to combine multiple forms of authentication into one, simple and easy to use dialog.

No more will you be required to select from a list of authentication lists, or be disappointed if a developer cannot implement a certain type. Rabbit automatically detects the type of input and redirects the credentials to the appropriate service. It currently supports Armor Games, Mousebreaker, Facebook, user id, and regular authentication. Furthermore, Rabbit will continue to include newer forms of authentication as they arise.

Find a problem? It's open source. You make the shots.

Because Rabbit is open source, it allows the community to build upon and shape the development of Rabbit. This allows for increased security and reliability, as many are able to see and audit the code.
